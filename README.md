## Project Overview

This project is a single-page React application with multiple routes demonstrating various tech solutions:

- **Generative AI** – Enterprise AI chatbots, custom LLMs, and automation workflows.
- **Managed Cloud Services** – Multi-cloud management, optimization, and support.
- **Modernization** – Legacy system upgrades, microservices, and UI/UX modernization.
- **Pinpoint Solutions** – Real-time analytics, targeted messaging, and secure communication.

The application also includes a responsive Navbar, Footer, dark/light mode support, and animated feature cards.

---

## Features

- Fully responsive design for desktop, tablet, and mobile.
- Interactive solution cards with hover effects.
- Dark mode support using Tailwind CSS.
- Semantic HTML structure with accessibility in mind.
- Clean, modular React components for each solution.
- Multi-route navigation using React Router v6.
- CTA sections with buttons for engagement.
- Cloud partner showcase for AI and cloud solutions.

---

## Technologies Used

- **React** – Frontend library for building UI components.
- **React Router v6** – Client-side routing.
- **Tailwind CSS** – Utility-first CSS framework for styling.
- **React Icons** – Icon library for modern icons.
- **JavaScript (ES6+)** – Dynamic interactivity.
- **Vite / Create React App** – Optional project scaffolding.
